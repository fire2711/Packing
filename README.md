# PackRight

This repository contains the project's Node.js environment, where the React/Vite app can be built and run.

To run it, follow these commands in the root folder:
```
cd packing-hci
npm install
npm run host
```

Then, open http://localhost:5173 on a browser to use the desktop interface. To use it on a mobile device connected to the same network, go to the URL given on the command line listed **Network**.
